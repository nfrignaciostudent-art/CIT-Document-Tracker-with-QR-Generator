Algorithm Info

IDEA (International Data Encryption Algorithm) is a symmetric-key block cipher using a 64-bit block and a 128-bit key, expanded into 52 subkeys over 8 full rounds plus an output transformation. Each round combines three algebraic operations: multiplication mod 65537, addition mod 65536, and bitwise XOR — making it resistant to differential and linear cryptanalysis.
